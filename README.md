# AIKO
A lightweight knowledge skeleton protocol for AI agents.

Not another sitemap.

Not another RSS feed.

Not another search index.

AIKO publishes knowledge structure, not content.

Its goal is simple:

Allow an AI agent to decide, within milliseconds, whether a website is worth reading further.

AIKO is intentionally minimal.

If a field does not help an AI decide whether to read the full document, it does not belong in AIKO.
